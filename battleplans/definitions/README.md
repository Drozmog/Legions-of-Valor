# Battleplan definitions

Create each battleplan here as a Godot `BattlePlanData` resource (`.tres`). Subfolders are supported and discovered automatically.

In Godot, right-click this folder, choose **New Resource**, select **BattlePlanData**, fill its fields, then save it as a `.tres` file. Assign the horizontal 3.5:2.5 card image to **Card Art**.

Suggested image folder: `res://battleplans/images/`.
